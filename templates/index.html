ZORA

WELCOME

Zora is a 'custom-built' business intelligence tool which analyzes sentiments from phrases provided to it by making use of meta's latest iteration of 'roBERTa-multilingual model. Zora's work flow is as follows:

Generate Dataset: scrape product-reviews from e-commerce sites. clean and format the data as csv. Dump dataset on amazon s3 bucket.

Fetch and Analyze: The generated dataset or any dataset from the s3 bucket is fetched. Sentiment analysis using meta's ROBERTA model is performed.

Data Insights: Once analysis has been performed, a graph of 'insights' is generated and displayed to be ued in conjuction with NPS data to derive Business Intellegence
